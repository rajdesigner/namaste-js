What is Virtual DOM?

The virtual DOM (VDOM) is a programming concept where an ideal, or “virtual”, representation of a UI is kept in memory and synced with the “real” DOM by a library such as ReactDOM. This process is called reconciliation.

What is React Fiber

Fiber is the new reconciliation engine in React 16. Its main goal is to enable incremental rendering of the virtual DOM.

Why do we need keys in React?When do we need Keys in react?

A “key” is a special string attribute you need to include when creating lists of elements in React. Keys are used in React to identify which items in the list are changed, updated, or deleted. In other words, we can say that keys are used to give an identity to the elements in the lists

Can we use index as a key in react?

We can use index as keys in react, but it is not recommended, It negatively impact performance and may cause issues with component state. As we know that React uses keys to identify we object is changed in real time and if we add new element with index as key, If the key is same as before React assumes that the DOM element represents the same component as before.

What is Props in react?

Props are arguments passed into React components. Props are passed to components via HTML attributes. Props are also how you pass data from one component to another, as parameters.
