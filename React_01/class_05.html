1. What is difference between Named Export, default Export and * as export?

Named Export

With Named Export we can import different components from a single file.

Default Export

With Default Export we can import a single Component from a file

* as export

With * as export we can import component and used (.) operator to access them as Object Properties.

2. What is importance of config.js file?

We can define our constants there, whose values don't have to change and it is a single source of truth.
